# Tech Stack

- React
- Styled-components
- React Hook Form
- themoviedb Api

## How to run the application

### How to start the server

Once this repo is cloned, open a terminal and run the command:

run:

#### "npm install"

change the .env.exmaple file to .env and put your api key for themoviedb Api

Finally, run:

#### "npm start"
